🧮 Simple Calculator (Tkinter)
A very simple calculator built with Python and Tkinter.
Currently, it supports basic addition with a clear and equals function.

🚀 Features
➕ Perform addition
🔢 Buttons for digits 0–9
🧹 Clear button to reset input
✅ Equals button to calculate result
🎨 Simple Tkinter GUI

Clone this repository:
git clone https://github.com/yourusername/tkinter-simple-calculator.git
cd tkinter-simple-calculator

Run the app:
python calculator.py
📂 Project Structure
├── calculator.py     
└── README.md       

📖 How to Use
Enter numbers using the on-screen buttons (0–9).
Press + to add another number.
Press = to see the result.
Press Clear to reset and start again.
